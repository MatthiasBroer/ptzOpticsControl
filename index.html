Welkom op bij PTZ Optics Controller

Dit is een project om te leren hoe ik een ptz camera kan bedieden
doormiddel van een webpagina en later ook doormiddel van een arduino.
Dit zodat ik mijn eigen controller kan bouwen en misschien op de markt
zou kunnen brengen als het een groot succes zou kunnen worden.
Vandaar ook dat dit een open source project is zodat er meer mensen
aan kunnen zitten en het een mooi product kan worden.
